# About Me Page Template

This is a simple About Me / Profile Page Template built using HTML and CSS. 

This project was created while learning web development from a YouTube tutorial and practicing basic webpage layout and styling.


## Features
* Simple and clean layout
* About Me section
* Profile image section
* Basic styling using CSS
* Beginner-friendly design
* Can be used as a personal portfolio template

## Technologies Used
* HTML
* CSS

## How to Use
1.	Download or clone this repository.
2.	Open index.html in your browser.
3.	Edit the text and images to add your own personal information.
4.	Customize colors and layout as you like.

## Future Improvements
* Make the page responsive for mobile devices
