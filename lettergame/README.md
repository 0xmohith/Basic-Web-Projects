# Letter Reaction Game

A simple and fun Letter Reaction Game built using HTML, CSS, and JavaScript.

 This project was created while learning web development from a YouTube tutorial.

 ## How the Game Works
* A random letter appears on the screen.
* The player must press the correct letter key on the keyboard within 3 seconds.
* If the correct key is pressed in time, the score increases.
* If the player presses the wrong key or runs out of time, the game ends.

## Technologies Used
* HTML
* CSS
* JavaScript

## Features
* Random letter generation
* 3-second timer
* Keyboard input detection
* Score tracker
* Game over on wrong key or timeout

## How to Run the Game
1.	Download or clone this repository.
2.	Open index.html in your browser.
3.	Press the correct letter before time runs out!
