# F1 CARS-HTML & CSS Project

## Overview
This is a simple front-end project based on Formula 1 cars created using only HTML and CSS.  
This project was made as part of my learning process to understand how websites are structured and styled.

## Features
* Responsive card layout
* Team-themed colors and backgrounds
* Image overlays with text
* Smooth hover effects
* Custom fonts and typography
* Simple and clean UI
* Simple Team Details page
* Simple responsive timeline design

## Technologies Used
* HTML
* CSS
* Google Fonts
* Responsive Design

## Purpose
The main purpose of this project is to understand HTML and CSS properly so that it becomes easier for me to move into Cybersecurity and Web Security later.   

By learning how websites are built, structured, and styled, it helps in understanding how websites can also be tested for vulnerabilities.

## What I Learned
* Basic HTML structure (head, body, div, sections)
* How CSS is used to style web pages
* Layout design using Flexbox and Grid
* Positioning elements (absolute, relative)
* How to overlay text on images
* Responsive design basics
* How front-end of a website is built

## How This Helps in Cybersecurity
Understanding HTML and CSS helps in cybersecurity because:

* Helps understand website structure during web app testing
* Helps in learning about XSS (Cross-Site Scripting)
* Helps in understanding forms and input fields
* Helps in understanding how login pages work
* Helps in basic web penetration testing
